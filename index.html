📡 LocalSend — Hızlı Dosya Transferi

📡 LocalSend Nedir?
Telefon ve bilgisayar aynı WiFi'ye bağlıyken internet gerekmeden anında dosya transferi. Hasta verileri dışarı çıkmaz. Tamamen ücretsiz.
Kurulum (1 Kez Yapılır)
💻 Bilgisayar (Windows)
1. Aşağıdan indirin
2. Kurun ve açın
3. WiFi'ye bağlı olun
4. Otomatik cihazları bulur
⬇️ Windows'a İndir
📱 Telefon
1. Store'dan indirin
2. Açın
3. Aynı WiFi'ye bağlı olun
4. Cihazlar birbirini bulur
🤖 Android 🍎 iPhone
Nasıl Kullanılır?
1
PC'de ⬆️ Dışa Aktar butonuna basın → JSON dosyası oluşur
2
PC'de LocalSend → Gönder → JSON dosyasını seçin → telefonu seçin → gönderin
3
⚠️ Önemli: Telefonda LocalSend bildirimine dokunmayın! Bunun yerine Kaydet seçeneğiyle İndirilenler klasörüne kaydedin
4
Bu uygulamada ⬇️ İçe Aktar → Dosyalar > İndirilenler klasöründen JSON dosyasını seçin
✅
İnternet gerekmez! LocalSend bildirimine değil, İçe Aktar butonuna basın.
